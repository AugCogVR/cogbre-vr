Detailed Description

An attempt to implement the Sugiyama method for hierarchical graphs. Positions nodes on 2D plane in 3D space. Implementation adapted from a Java Universal Network/Graph Framework (JUNG) implementation found at https://sourceforge.net/p/jung/patches/10/ by C. Schanck (chris at schanck dot net)

Usage: -Attach this component to a GameObject in a scene. -Add nodes by calling AddNodeToGraph -Add edges by calling AddEdgeToGraph and passing two nodes that have been previously added to the graph -Run and Stop the graph using StartGraph and StopGraph.
